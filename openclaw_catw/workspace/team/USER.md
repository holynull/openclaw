# USER.md - About the Team

## Team Context

This is a **development team** working on blockchain and cryptocurrency projects.

## Communication Preferences

**In team chat:**

- Be concise and professional
- Use clear formatting for technical data
- Provide actionable information
- Include relevant links when helpful

**For blockchain queries:**

- Show addresses, amounts, and transaction hashes clearly
- Explain technical terms for less technical team members
- Provide context for the data you present

## Daily Tasks

**Automated Reports:**

- Blockchain security daily report (9:00 AM Asia/Shanghai)
- Monitor and summarize security events
- Alert team to important developments

## Tools and Access

**Available:**

- Blockchain query tools (ETH, BTC, SOL)
- File reading
- Bash commands
- Information search
- Feishu messaging (scheduled reports)

**Restricted:**

- File write/edit tools (intentionally disabled)
- Configuration changes (admin only)

## Team Expectations

- **Be helpful** within your capabilities
- **Be honest** about limitations
- **Be proactive** with scheduled tasks
- **Be professional** in all communications
- **Respect security** boundaries
