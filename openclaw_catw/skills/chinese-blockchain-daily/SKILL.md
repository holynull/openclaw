# 中文区块链日报生成 Skill

## 任务目标

生成中文区块链日报并发送到飞书群，包含最少15条新闻，覆盖重大事件、融资、市场、监管、安全等多个维度。

## 1️⃣ 数据源策略（降级方案）

### 优先方案（web_fetch，免费）

使用 `web_fetch` 工具获取以下RSS源：

- https://www.chainfeeds.xyz/rss
- https://www.panewslab.com/zh/rss/foryou.xml
- https://www.panewslab.com/zh/rss/newsflash.xml

### 降级方案（仅当web_fetch全部失败或提取新闻<10条时）

使用 `web_search` 工具多次搜索不同关键词组合：

- "区块链" (count=10)
- "加密货币 融资" (count=5)
- "比特币 以太坊" (count=5)
- "Web3 项目" (count=5)
- "区块链 监管" (count=5)
- "加密货币 安全 黑客" (count=5)

时间范围：freshness="day"

## 2️⃣ 内容要求（必须满足，宁可多不可少）

- **最少15条新闻**（🔥重大事件≥3条、💰融资≥2条、📊市场≥3条、其他类型各≥2条）
- 每条新闻包含：标题、简述（30-50字）、**完整URL**
- **⚠️ 重要：不要过度过滤！只要与区块链/加密货币相关，都应保留**
- 允许包含：项目更新、技术文章、市场分析、行业动态、教育内容
- 仅排除：完全无关内容、纯广告
- 如果web_search结果<15条，继续添加新关键词搜索（如"NFT" "DeFi" "Web3游戏" "元宇宙"）

## 3️⃣ 报告格式（飞书文档Markdown）

```markdown
# 📰 中文区块链日报 | {日期}

---

## 🔥 重大事件 (N条)

### 标题1

📄 简述内容（重点突出，30-50字）  
🔗 [查看详情](完整URL)

### 标题2

📄 简述内容  
🔗 [查看详情](完整URL)

---

## 💰 融资与项目 (N条)

### 项目名 - 融资$XXX万

📄 简述  
🔗 [详情](URL)

---

## 📊 市场与技术 (N条)

### 标题

📄 简述  
🔗 [来源](URL)

---

## 🏛️ 监管动态

（如有内容时才显示此节）

---

## 🛡️ 安全警示

（如有内容时才显示此节）

---

**📊 今日统计**：共X条要闻 | 数据源：Y家
```

### 格式要求

- 使用标准Markdown语法（# ## ### 标题，- 列表）
- 每条新闻用 ### 三级标题
- 简述和链接各占一行
- 标题不要加粗符号 \*\*，直接使用 ### 即可
- 链接必须是完整URL，格式：[文字](URL)

## 4️⃣ 保存并发送

### 步骤1：创建飞书文档

使用 `feishu_doc` 工具：

```
action: 'create'
title: '📰 中文区块链日报 | {日期}'
grant_to_requester: true
```

### 步骤2：写入报告内容

使用 `feishu_doc` 工具：

```
action: 'write'
doc_token: {从步骤1获取的document_id}
content: {完整日报Markdown内容，使用上述格式}
```

### 步骤3：生成摘要并发送到群

使用 `bash` 工具创建摘要文件：

```bash
SUMMARY_FILE="/tmp/daily-summary-$(date +%Y%m%d).txt"
cat > "$SUMMARY_FILE" << 'EOFSUM'
📊 今日要闻：{X}条
• 🔥 重大事件：{N}条
• 💰 融资项目：{N}条
• 📊 市场动态：{N}条

📄 完整报告文档：
{从步骤1获取的飞书文档URL}

数据源：{Y}家
EOFSUM
```

然后使用 `send_feishu_file_content` 工具：

```
filePath: 摘要文件路径
chatId: 'oc_53d1a541f08d2d9f2e8c3c79a1f12fc3'
title: '📰 中文区块链日报'
maxLength: 1000
```

## ⚠️ 注意事项

1. 所有步骤必须成功完成
2. 如果新闻数<15条，继续搜索更多关键词
3. 确保文档中所有URL完整可点击
